# Project-Language-Alphabet
This project validates lines of code with valid () [] {}.

Accepted language - language alphabet: <{[())}>

Example of inputs:

[](){} - OK - Válido
[)]{} - Não - Inválido
[(]){}<> - Não - Inválido
(((((([])))))) - OK - Válido
{{[[()]()]}} - OK - Válido
{[[([)]]]} - Não - Inválido
<{(())}> - OK - Válido
{<[]()[[]]<>>} - OK – Válido
Escreva aqui os alfabetos de linguagem. Ex: <{[()]}>
